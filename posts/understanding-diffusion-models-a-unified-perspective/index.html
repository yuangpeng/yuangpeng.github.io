Introduction: Generative Models#

给定某一分布的观测数据 ${\mathbf{x}_1,\mathbf{x}_2,\dots,\mathbf{x}_n}$，生成模型的目标是学习建模其真实分布 $p(\mathbf{x})$。一旦“学会”，我们即可从估计分布中采样新的样本点，也可以使用学习到的模型估计观测数据或者采样数据的似然。

目前生成模型主要分为如下四类：

  • GANs 以对抗的方式进行训练，generator 学习条件分布，然后通过 ancestral sampling 采样出生成数据，discriminator 评估模型生成数据分布和真实分布的接近程度。GANs 采取对抗训练的方式，会导致其训练不稳定，生成图片的具有较弱的多样性。
  • likelihood-based model 学习一个模型，为观测数据分配一个最高的 likelihood，即 $\theta^*=\arg\max \prod p_\theta(\mathbf{x}_i)$。该类模型包括 Flow-based models、VAEs 等。
  • energy-based model 学习任意的能量函数（energy function），然后对其归一化。
  • score-based generative model 不直接去学 energy function，而是去学 score of the energy-based model （也就是 score function）。

diffusion model 既可以从 likelihood-based 的视角解释，也可以从 score-based 的视角去解释。

ELBO, VAE and Hierarchical VAE#

在许多模式下，我们可以认为观测数据 $\mathbf{x}$ 是由一个相关的“不可见的”隐变量（latent variable）代表或产生的，该隐变量可用随机变量 $\mathbf{z}$ 来表示，并假设其服从一个先验分布 $p(\mathbf{z})$。我们不直接建模 $p_\theta(\mathbf{x})$，而是建模 $p_\theta(\mathbf{x},\mathbf{z})$ 从而间接建模 $p_\theta(\mathbf{x})$，原因可能有：

  • 神经网络的对于分布表达能力有限，一般来说网络输出均值和方差来表示高斯分布，无法表达过于复杂分布。
  • 如果建模了数据 $\mathbf{x}$ 和隐变量 $\mathbf{z}$ 的关系，对于一个已知先验分布的隐变量，我们可以使用 ancestral sampling 对 $p_\theta(\mathbf{x})$ 进行采样。
  • 一般来说隐变量的维度比数据维度小，学习低维隐变量也可以被看作是一种压缩，并有可能发现观测数据的语义结构。

Evidence Lower Bound#

我们用一个联合分布 $p_\theta(\mathbf{x}, \mathbf{z})$ 对观测数据和隐变量建模，likelihood-based 方法的目标是最大化所有观测数据 $\mathbf{x}$ 的似然 $p_\theta(\mathbf{x})$。这里有两种方式可以将联合分布转换为 $p_\theta(\mathbf{x})$：

  • 对隐变量 $\mathbf{z}$ 积分求的边缘分布：

$$ p_\theta(\mathbf{x})=\int p_\theta(\mathbf{x}, \mathbf{z}) \mathrm{d} \mathbf{z} $$

  • 运用条件概率的链式法则：

$$ p_\theta(\mathbf{x})=\frac{p_\theta(\mathbf{x},\mathbf{z})}{p_\theta(\mathbf{z} \mid \mathbf{x})} $$

直接计算优化似然 $p_\theta(\mathbf{x})$ 是非常困难的，因为要不就得对所有的隐变量积分，要不就得已知后验 $p_\theta(\mathbf{z} \mid \mathbf{x})$。在引入了隐变量 $\mathbf{z}$ 之后，我们的目标是学习隐变量和我们观测数据的潜在关系。换句话说我们可以再引入一个变分后验 $q_\phi(\mathbf{z}\mid \mathbf{x})$，优化变分后验的参数 $\phi$，使之与后验分布 $p_\theta(\mathbf{z} \mid \mathbf{x})$ 完全匹配，即通过最小化它们的 KL divergence（最好为零）来实现。得到了两者潜在关系 $p_\theta(\mathbf{x},\mathbf{z})$ 和 $q_\phi(\mathbf{z}\mid \mathbf{x})$ 后，我们便可利用链式法则得到数据的估计真实分布 $p_\theta(\mathbf{x})$。此时我们就有两个优化目标：1）优化变分后验 $q_\phi(\mathbf{z}\mid \mathbf{x})$ 的参数 $\phi$，最小化和估计真实后验 $p_\theta(\mathbf{z} \mid \mathbf{x})$ 的 KL divergence；2）优化参数 $\theta$ 使得 log likelihood $\log p_\theta(\mathbf{x})$ 最大。此时有一个最大的问题是我们无法计算 KL divergence。

我们引入一个 Evidence Lower Bound (ELBO)，名字中 evidence 表示所有观测数据的 log likelihood $\log p_\theta(\mathbf{x})$，即 ELBO 是 log likelihood 的下界。通过优化 ELBO，我们可以做到同时优化 log likelihood 和最小化 KL divergence。具体来说，当参数 $\phi$ 被完全优化，ELBO 会最小化 KL divergence，并且等于 evidence；当参数 $\theta$ 被优化，优化 ELBO 即提高了 log likelihood 的下界，在参数 $\phi$ 被完全优化的情况下，等价于优化了 log likelihood；当参数 $\theta$ 被优化后，又需要重新优化参数 $\phi$ ……

$$ \text{ELBO}=\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p_\theta(\mathbf{x}, \mathbf{z})}{q_{\mathbf{\phi}}(\mathbf{z} \mid \mathbf{x})}\right] $$

下面的推导过程深刻揭示了 ELBO 为什么能够同时优化上面两个目标，并且解决无法计算 KL divergence 的问题。从下面的推导可以看出 $\text{evidence} = \text{ELBO} + D_\mathrm{KL}(q_\phi(\mathbf{z} \mid \mathbf{x}) | p_\theta(\mathbf{z} \mid \mathbf{x}))$。evidence $\log p_\theta(\mathbf{x})$ 相对于参数 $\phi$ 来说始终是一个常数，因为无论从条件概率的链式法则推导还是对隐变量 $\mathbf{z}$ 积分求的边缘分布，$p_\theta(\mathbf{x})$ 的计算都不依赖于 $\phi$。即 ELBO 和 KL divergence 的和对于参数 $\phi$ 来说为常数，因此对于 $\phi$ 的优化会导致 ELBO 和 KL divergence 此消彼长。因此优化 $\phi$ 去最大化 ELBO 可以间接最小化 KL divergence。并且在 $\phi$ 基本完全优化的情况，KL divergence 为0，因此优化 $\theta$ 提高 ELBO，并又使得 KL divergence 大于 0，从而间接优化 log likelihood。

$\begin{aligned} \log p_\theta(\mathbf{x}) &=\log p_\theta(\mathbf{x}) \int q_\phi(\mathbf{z} \mid \mathbf{x}) \mathrm{d} \mathbf{z} & &\text { (Multiply by } 1=\int q_{\mathbf{\phi}}(\mathbf{z} \mid \mathbf{x}) \mathrm{d} \mathbf{z}) \ &=\int q_\phi(\mathbf{z} \mid \mathbf{x})(\log p_\theta(\mathbf{x})) \mathrm{d} \mathbf{z} & & \text { (Bring evidence into integral) } \ &=\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}[\log p_\theta(\mathbf{x})] & & \text { (Definition of Expectation) } \ &=\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p_\theta(\mathbf{x}, \mathbf{z})}{p_\theta(\mathbf{z} \mid \mathbf{x})}\right] & & \text { (Chain Rule of Probability) } \ &=\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p_\theta(\mathbf{x}, \mathbf{z}) q_\phi(\mathbf{z} \mid \mathbf{x})}{p_\theta(\mathbf{z} \mid \mathbf{x}) q_\phi(\mathbf{z} \mid \mathbf{x})}\right] & &\text { (Multiply by } 1=\frac{q_\phi(\mathbf{z} \mid \mathbf{x})}{q_\phi(\mathbf{z} \mid \mathbf{x})}) \ &=\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p_\theta(\mathbf{x}, \mathbf{z})}{q_\phi(\mathbf{z} \mid \mathbf{x})}\right]+\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{q_\phi(\mathbf{z} \mid \mathbf{x})}{p_\theta(\mathbf{z} \mid \mathbf{x})}\right] & & \text { (Split the Expectation) } \ &=\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p_\theta(\mathbf{x}, \mathbf{z})}{q_\phi(\mathbf{z} \mid \mathbf{x})}\right]+D_{\mathrm{KL}}(q_\phi(\mathbf{z} \mid \mathbf{x}) | p_\theta(\mathbf{z} \mid \mathbf{x})) \text { }& & \text { (Definition of KL Divergence) } \ & \geq \mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p_\theta(\mathbf{x}, \mathbf{z})}{q_\phi(\mathbf{z} \mid \mathbf{x})}\right] & &\text { (KL Divergence always } \geq 0) \end{aligned}$

Variational Autoencoders#

在 VAE 中，在 ELBO 上联合优化参数 $\phi$ 和 $\theta$ 从而间接最小化 KL divergence 并且最大化似然，这种方法叫做 variational，因为我们在以 $\phi$ 为参数的所有潜在后验分布中优化找到最佳的 $q_\phi(\mathbf{z} \mid \mathbf{x})$。它被称为 autoencoder 是因为它让人联想到传统的 autoencoder model，即输入数据在经过中间的 bottleneck representation 后被用来训练来预测本身。为了更清楚阐明其与传统 autoencoder 模型的关系，我们进一步对 ELBO 推导：

$$ \begin{aligned} \mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p_\theta(\mathbf{x}, \mathbf{z})}{q_\phi(\mathbf{z} \mid \mathbf{x})}\right]&=\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p_{\mathbf{\theta}}(\mathbf{x} \mid \mathbf{z}) p(\mathbf{z})}{q_\phi(\mathbf{z} \mid \mathbf{x})}\right] & & \text{ (Chain Rule of Probability) } \ &=\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log p_{\mathbf{\theta}}(\mathbf{x} \mid \mathbf{z})\right]+\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log \frac{p(\mathbf{z})}{q_\phi(\mathbf{z} \mid \mathbf{x})}\right] & & \text { (Split the Expectation) }\ &=\underbrace{\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log p_{\mathbf{\theta}}(\mathbf{x} \mid \mathbf{z})\right]}{\text {reconstruction term }}-\underbrace{D{\mathrm{KL}}\left(q_{\mathbf{\phi}}(\mathbf{z} \mid \mathbf{x}) | p(\mathbf{z})\right)}_{\text {prior matching term }} & & \text { (Definition of KL Divergence) } \end{aligned} $$

在这种情况下，学习到的 bottleneck distribution $q_\phi(\mathbf{z} \mid \mathbf{x})$ 可以看作成是 encoder，它将输入转化成隐变量的分布；同时我们学习了一个确定性的函数 $p_{\mathbf{\theta}}(\mathbf{x} \mid \mathbf{z})$，将给定的隐变量转换为观测数据 $\mathbf{x}$，这可以被解释为一个解码器。ELBO 最终推导出的两项都有比较直觉的描述：第一个项衡量 decoder 从我们的变分分布中重建 $\mathbf{x}$ 的可能性；第二项衡量学到的变分分布与隐变量的先验分布的相似程度。最大化 ELBO 等同于最大化其第一项和最小化其第二项，但是值得注意的是，最大化 ELBO 并不是单独去最大化第一项和最小化第二项，因为这两项都是和 $\phi$ 有联系的，优化 $\phi$ 在直觉上我们需要让第一项尽可能大且第二项尽可能小。实际上我们不能让 $D_{\mathrm{KL}}\left(q_{\mathbf{\phi}}(\mathbf{z} \mid \mathbf{x}) | p(\mathbf{z})\right)$ 接近 0，优化 ELBO 真正是需要让 $D_{\mathrm{KL}}(q_\phi(\mathbf{z} \mid \mathbf{x}) | p_\theta(\mathbf{z} \mid \mathbf{x}))$ 接近 0。

一个复杂的分布其实可以表示成多个高斯分布的和，因此理论上，我们其实只要求 decoder $p_\theta(\mathbf{x} \mid \mathbf{z})$ 为高斯分布，先验分布 $p(\mathbf{z})$ 的复杂程度决定了混合高斯分布的复杂程度（也就是无数多个高斯分布的和）。$p_\theta(\mathbf{x})= \int p_\theta(\mathbf{x} \mid \mathbf{z})p(\mathbf{z})\mathrm{d}\mathbf{z}$ 为一个混合高斯分布，实际上我们仅仅假设隐变量 $\mathbf{z}$ 服从简单的高斯分布，$p_\theta(\mathbf{x})$ 就可以表达出足够复杂的分布了，但对先验分布的假设也是限制 VAE 模型表达能力的瓶颈。对于变分后验 $q_\phi(\mathbf{z} \mid \mathbf{x})$，我们一般建模成有对角协方差矩阵的多维高斯分布（也就是各维随机变量相互独立）$q_{\mathbf{\phi}}(\mathbf{z} \mid \mathbf{x})=\mathcal{N}\left(\mathbf{z} ; {\mu}\phi(\mathbf{x}), \mathbf{\sigma}{\mathbf{\phi}}^2(\mathbf{x}) \mathbf{I}\right)$。但是对于估计的真实后验 $p_\theta(\mathbf{z} \mid \mathbf{x}) = \frac{p_\theta(\mathbf{x} \mid \mathbf{z})p(\mathbf{z})}{p_\theta(\mathbf{x})}$ 就不再是一个简单的高斯分布，因此我们对于变分后验的建模能力限制了整个模型的表达能力。

对于优化目标 ELBO，我们可以解析计算其 KL divergence 项，并且可以使用蒙特卡洛估计重建项（我的理解是直接用蒙特卡洛估计原始的 ELBO 和下面分成重建项与 KL divergence 项本质上是一样的，但是分成两项可以减少蒙特卡洛估计的方差）：

$$ \underset{\mathbf{\phi}, \mathbf{\theta}}{\arg \max } \left(\mathbb{E}{q\phi(\mathbf{z} \mid \mathbf{x})}\left[\log p_{\mathbf{\theta}}(\mathbf{x} \mid \mathbf{z})\right]-D_{\mathrm{KL}}\left(q_\phi(\mathbf{z} \mid \mathbf{x}) | p(\mathbf{z})\right) \right)\approx \underset{\mathbf{\phi}, \mathbf{\theta}}{\arg \max } \left(\sum_{l=1}^L \log p_{\mathbf{\theta}}\left(\mathbf{x} \mid \mathbf{z}^{(l)}\right)-D_{\mathrm{KL}}\left(q_{\mathbf{\phi}}(\mathbf{z} \mid \mathbf{x}) | p(\mathbf{z})\right)\right) $$

对于数据集中每个观测数据 $\mathbf{x}$，隐变量 ${ \mathbf{z}^{(l)} }{l=1}^{L}$ 从 $q\phi(\mathbf{z} \mid \mathbf{x})$ 采样。然而 $\mathbf{z}^{(l)}$ 是一个随机采样的过程，是不可微分的，如此我们便无法传递梯度去更新 $\phi$。解决方式是采用 reparameterization trick，随机变量 $\mathbf{z} = \mu_\phi(\mathbf{x}) + \sigma_\phi(\mathbf{x}) \odot \epsilon \quad \text{ with } \epsilon \sim \mathcal{N}(\mathbf{0},\mathbf{I})$。计算图如下所示，利用蒙特卡洛估计和 reparameterization trick，梯度随着红线去优化参数 $\phi$ 和 $\theta$。

Hierarchical VAE#

HVAE 是将 VAE 拓展到多层次隐变量，在这种情况下，隐变量被解释成由更高层次、更抽象的隐变量生成的隐变量。在上一节提到过，$p_\theta(\mathbf{x})= \int p_\theta(\mathbf{x} \mid \mathbf{z})p(\mathbf{z})\mathrm{d}\mathbf{z}$ 是一个混合高斯分布，$p(\mathbf{z})$ 是决定这个混合高斯分布表达能力的瓶颈。由于我们假设 $p(\mathbf{z})$ 为高斯分布，也可以反过来看假设 $p_\theta(\mathbf{x} \mid \mathbf{z})$ 为高斯分布是限制这个混合高斯分布表达能力的瓶颈。在神经网络中，我们可以用 relu 函数叠加逼近任何函数，理论上该混合高斯分布也已经可以表示足够复杂的分布。但我们知道，deeper 的网络比 flatter 的网络表达能力更强，也就是更深的网络非线性表达能力更强，在这里我们也可以同样利用该思想，不如我们也增加混合高斯分布的深度。在具有 $T$ 层的 HVAE 中，每个隐变量都被允许依赖于之前所有的隐变量，但我们这里只专注于 Markovian HVAE (MHVAE)，也就是 encode $\mathbf{z}t$ 只依赖于 $\mathbf{z}{t-1}$，decode $\mathbf{z}{t-1}$ 只依赖于 $\mathbf{z}t$。在这种情况下，我们增加混合高斯分布深度即 $p\theta(\mathbf{x})= \int p\theta(\mathbf{x} \mid \mathbf{z})p(\mathbf{z})\mathrm{d}\mathbf{z}=\int p_{\mathbf{\theta}}\left(\mathbf{x} \mid \mathbf{z}1\right) \prod{t=2}^T p_{\mathbf{\theta}}\left(\mathbf{z}{t-1} \mid \mathbf{z}t\right) p\left(\mathbf{z}T\right) \mathrm{d}\mathbf{z}{1:T}$。我们在上一节提到过估计的真实后验 $p\theta(\mathbf{z} \mid \mathbf{x}) = \frac{p\theta(\mathbf{x} \mid \mathbf{z})p(\mathbf{z})}{p_\theta(\mathbf{x})}$ 可能是一个复杂的分布，而我们仅仅用高斯分布 $q_\phi(\mathbf{z} \mid \mathbf{x})$ 去近似真实后验，那必然会存在一定的误差，同理我们也可以增加高斯分布深度即 $p_\theta(\mathbf{z} \mid \mathbf{x}) \approx q_\phi(\mathbf{z} \mid \mathbf{x}) \approx \int\prod_{t=2}^{T} q_\phi(\mathbf{z}t \mid \mathbf{z}{t-1})q_\phi(\mathbf{z}1 \mid \mathbf{x})\mathrm{d}\mathbf{z}{1:T-1}$。

在数学上，MHVAE 的隐变量和数据的联合分布与变分后验可以表示为：

$$ \begin{aligned} p_\theta\left(\mathbf{x}, \mathbf{z}{1: T}\right)&=p\left(\mathbf{z}T\right) p{\mathbf{\theta}}\left(\mathbf{x} \mid \mathbf{z}1\right) \prod{t=2}^T p{\mathbf{\theta}}\left(\mathbf{z}{t-1} \mid \mathbf{z}t\right) &\q{\mathbf{\phi}}\left(\mathbf{z}{1: T} \mid \mathbf{x}\right)&=q_{\mathbf{\phi}}\left(\mathbf{z}1 \mid \mathbf{x}\right) \prod{t=2}^T q_{\mathbf{\phi}}\left(\mathbf{z}t \mid \mathbf{z}{t-1}\right) \end{aligned} $$

因此可以类似得写出 ELBO 的表达式

$$ \log p_\theta(\mathbf{x}) = \mathbb{E}{q\phi\left(\mathbf{z}{1: T} \mid \mathbf{x}\right)}\left[\log \frac{p\theta\left(\mathbf{x}, \mathbf{z}{1: T}\right)}{q{\mathbf{\phi}}\left(\mathbf{z}{1: T} \mid \mathbf{x}\right)}\right] +D{\mathrm{KL}}(q_\phi(\mathbf{z}{1:T} \mid \mathbf{x}) | p\theta(\mathbf{z}_{1:T} \mid \mathbf{x})) $$

我们可以将联合分布和变分后验带入 ELBO，得到如下表达式。现在的表达式看起来是非常复杂，当深入到下一节后，这个复杂的表达式可以分解成各个可解释的部分。

$$ \mathbb{E}{q{\mathbf{\phi}}\left(\mathbf{z}{1: T} \mid \mathbf{x}\right)}\left[\log \frac{p\theta\left(\mathbf{x}, \mathbf{z}{1: T}\right)}{q{\mathbf{\phi}}\left(\mathbf{z}{1: T} \mid \mathbf{x}\right)}\right]=\mathbb{E}{q_{\mathbf{\phi}}\left(\mathbf{z}{1: T} \mid \mathbf{x}\right)}\left[\log \frac{p\theta\left(\mathbf{z}T\right) p{\mathbf{\theta}}\left(\mathbf{x} \mid \mathbf{z}1\right) \prod{t=2}^T p_{\mathbf{\theta}}\left(\mathbf{z}{t-1} \mid \mathbf{z}t\right)}{q{\mathbf{\phi}}\left(\mathbf{z}1 \mid \mathbf{x}\right) \prod{t=2}^T q{\mathbf{\phi}}\left(\mathbf{z}t \mid \mathbf{z}{t-1}\right)}\right] $$

Variational Diffusion Models#

从 VAE 的角度切入看待的 diffusion model 其实只是在 MHVAE 上有三点假设：

  • 隐变量和观测数据的维度完全一致，也就是在 diffusion model 中，不再对数据进行压缩。
  • encoder $q_\phi$ 不再需要学习，而是被预先定义为一个线性高斯模型。
  • encoder 的高斯分布参数随时间变化，在最终时间步长 $T$ 的隐变量分布是一个标准的高斯分布。

回顾 VAE，我们有观测数据和隐变量的分布 $p(\mathbf{x})$ 和 $p(\mathbf{z})$，我们要做的事情是联合训练 $q_\phi(\mathbf{z} \mid \mathbf{x})$ 和 $p_\theta(\mathbf{x} \mid \mathbf{z})$。是给定 $p_\theta(\mathbf{x} \mid \mathbf{z})$ 的情况下，$q_\phi(\mathbf{z} \mid \mathbf{x})$ 要去匹配 $p_\theta(\mathbf{x} \mid \mathbf{z})$，然后调整 $\theta$ 提高 ELBO，即提高 likelihood，如此进行两者联合训练。根据贝叶斯公式，我们有 $p(\mathbf{x} \mid \mathbf{z})p(\mathbf{z})=p(\mathbf{z} \mid \mathbf{x})p(\mathbf{x})$，在 $p(\mathbf{z})$ 和 $p(\mathbf{x})$ 是确定的情况下，$p(\mathbf{x} \mid \mathbf{z})$ 和 $p(\mathbf{z} \mid \mathbf{x})$ 的解是一一对应的，并且解不唯一。而训练 VAE 其实就是在那个硕大的解空间去寻找一组解。MHVAE 主要解决了 $q_\theta$ 和 $q_\phi$ 表达能力不强的问题。因为在 VAE 中一旦定义好 decoder $p_\theta(\mathbf{x} \mid \mathbf{z})$ 是高斯分布，那么能表达的 $p_\theta(\mathbf{x})$ 就是有限的，并且根据公式 $p(\mathbf{z} \mid \mathbf{x}) = \frac{p(\mathbf{x} \mid \mathbf{z})p(\mathbf{z})}{p(\mathbf{x})}$ encoder 大概率不是一个简单的高斯分布，因此对于 decoder 就只能在高斯分布的解空间找一个近似解。在 MHVAE 中我们还没有好好利用马尔可夫链的性质，联合训练两个网络去解空间找解是一个比较麻烦的事情，但是我们可以定义一条平稳马尔可夫链，则可不去学习 $\phi$。虽然可能我们手工构造的平稳马尔可夫链不是最优解，但是可以大大简化训练难度，即我们只需要根据我们定义的 encoder 去学习 decoder $p_\theta$。

现在将上述三个假设写成数学的表达形式。第一个假设是数据和隐变量是同一维度，即可以都用 $\mathbf{x}_t$ 来表示，其中 $t=0$ 代表观测数据，$t \in [1,T]$ 代表用 $t$ 为索引层次的隐变量，因此 VDM 中的变分后验可以写成：

$$ q\left(\mathbf{x}_{1: T} \mid \mathbf{x}0\right)=\prod{t=1}^T q\left(\mathbf{x}t \mid \mathbf{x}{t-1}\right) $$

第二个假设是 $q_\phi$ 不再需要学习，我们直接定义为线性高斯模型，即高斯分布的均值 $\mu_t(\mathbf{x}t) = \sqrt{\alpha_t} \mathbf{x}{t-1}$ 和方差 $\Sigma_t\left(\boldsymbol{x}_t\right)=\left(1-\alpha_t\right) \mathbf{I}$ 都是提前设置的超参数：

$$ q\left(\mathbf{x}t \mid \mathbf{x}{t-1}\right)=\mathcal{N}\left(\mathbf{x}t ; \sqrt{\alpha_t} \mathbf{x}{t-1},\left(1-\alpha_t\right) \mathbf{I}\right) $$

第三个假设是因为我们可以构造好 $\alpha_t$，使得最终的 $p(\mathbf{x}_T)$ 是一个标准的高斯分布。因此观测数据和隐变量的联合分布可以写成：

$$ p_\theta\left(\mathbf{x}{0: T}\right)=p\left(\mathbf{x}T\right) \prod{t=1}^T p{\mathbf{\theta}}\left(\mathbf{x}_{t-1} \mid \mathbf{x}_t\right) \quad \text{ where } p\left(\mathbf{x}_T\right)=\mathcal{N}\left(\mathbf{x}_T ; \mathbf{0}, \mathbf{I}\right) $$

总的来说，这三个假设所描述的是，随着时间的推移，逐渐通过添加高斯噪声来逐步破坏图像，直到它变成纯高斯噪声。从视觉上看，这个过程如下所示：

由于编码和解码（正向和逆向）都是马尔可夫链，因此我们有 $q(\mathbf{x}t\mid \mathbf{x}{t-1}) = q(\mathbf{x}t\mid \mathbf{x}{t-1}, \mathbf{x}_0)$，根据 Bayes rule，可以得到 $q\left(\mathbf{x}t \mid \mathbf{x}{t-1}, \mathbf{x}0\right)=\frac{q\left(\mathbf{x}{t-1} \mid \mathbf{x}_t, \mathbf{x}_0\right) q\left(\mathbf{x}_t \mid \mathbf{x}0\right)}{q\left(\mathbf{x}{t-1} \mid \mathbf{x}_0\right)}$。类似 MHVAE 中的 ELBO 表达式，结合上面 Bayes rule，我们可以写出：

$$ \begin{aligned}\log p_\theta(\mathbf{x}0) &\geq \underbrace{\mathbb{E}{q\left(\mathbf{x}1 \mid \mathbf{x}0\right)}\left[\log p{\mathbf{\theta}}\left(\mathbf{x}0 \mid \mathbf{x}1\right)\right]}{\text {reconstruction term }}-\underbrace{D{\mathrm{KL}}\left(q\left(\mathbf{x}T \mid \mathbf{x}0\right) | p\left(\mathbf{x}T\right)\right)}{\text {prior matching term }}-\sum{t=2}^T \underbrace{\mathbb{E}{q\left(\mathbf{x}t \mid \mathbf{x}0\right)}\left[D{\mathrm{KL}}\left(q\left(\mathbf{x}{t-1} \mid \mathbf{x}t, \mathbf{x}0\right) | p{\mathbf{\theta}}\left(\mathbf{x}{t-1} \mid \mathbf{x}t\right)\right)\right]}{\text {denoising matching term }}\ & \approx \mathbb{E}{q\left(\mathbf{x}_1 \mid \mathbf{x}0\right)}\left[\log p{\mathbf{\theta}}\left(\mathbf{x}_0 \mid \mathbf{x}_1\right)\right]

-\mathbb{E}{t \sim \mathcal{U}(2, T)}\left[\mathbb{E}{q\left(\mathbf{x}t \mid \mathbf{x}0\right)}\left[D{\mathrm{KL}}\left(q\left(\mathbf{x}{t-1} \mid \mathbf{x}t, \mathbf{x}0\right) | p{\mathbf{\theta}}\left(\mathbf{x}{t-1} \mid \mathbf{x}t\right)\right)\right] \right] \ & \approx -\mathbb{E}{t \sim \mathcal{U}(1, T)}\left[\mathbb{E}_{q\left(\mathbf{x}t \mid \mathbf{x}0\right)}\left[D{\mathrm{KL}}\left(q\left(\mathbf{x}{t-1} \mid \mathbf{x}t, \mathbf{x}0\right) | p{\mathbf{\theta}}\left(\mathbf{x}{t-1} \mid \mathbf{x}_t\right)\right)\right] \right] \end{aligned} $$

  • $\mathbb{E}_{q\left(\mathbf{x}_1 \mid \mathbf{x}0\right)}\left[\log p\theta\left(\mathbf{x}_0 \mid \mathbf{x}_1\right)\right]$ 可以被解释成重建项，该项可以用蒙特卡洛估计期望。
  • $D_{\mathrm{KL}}\left(q\left(\mathbf{x}_T \mid \mathbf{x}_0\right) | p\left(\mathbf{x}_T\right)\right)$ 代表最终加噪图片的分布与标准高斯先验分布的接近程度。在我们的假设下这一项为 0。
  • $\mathbb{E}{q\left(\mathbf{x}t \mid \mathbf{x}0\right)}\left[D{\mathrm{KL}}\left(q\left(\mathbf{x}{t-1} \mid \mathbf{x}t, \mathbf{x}0\right) | p\theta\left(\mathbf{x}{t-1} \mid \mathbf{x}t\right)\right)\right]$ 代表我们期望学习到的去噪 $p\theta(\mathbf{x}{t-1} \mid \mathbf{x}t)$ 步骤和 ground truth $D\mathrm{KL}(q(\mathbf{x}_{t-1} \mid \mathbf{x}_t, \mathbf{x}_0)$ 越接近越好。

根据 Bayes rule，我们有：

$$ q\left(\mathbf{x}_{t-1} \mid \mathbf{x}_t, \mathbf{x}_0\right)=\frac{q\left(\mathbf{x}t \mid \mathbf{x}{t-1}, \mathbf{x}0\right) q\left(\mathbf{x}{t-1} \mid \mathbf{x}_0\right)}{q\left(\mathbf{x}_t \mid \mathbf{x}_0\right)} $$

其中我们已知 $q\left(\mathbf{x}t \mid \mathbf{x}{t-1}, \mathbf{x}_0\right)=q\left(\mathbf{x}t \mid \mathbf{x}{t-1}\right)=\mathcal{N}\left(\mathbf{x}t ; \sqrt{\alpha_t} \mathbf{x}{t-1},\left(1-\alpha_t\right) \mathbf{I}\right)$。利用 reparameterization trick，对于采样 $\mathbf{x}_t \sim q(\mathbf{x}t \mid \mathbf{x}{t-1})$，可以重写为：

$$ \begin{aligned}\mathbf{x}t&=\sqrt{\alpha_t} \mathbf{x}{t-1}+\sqrt{1-\alpha_t} \epsilon_{t-1}^\&=\sqrt{\alpha_t}\left(\sqrt{\alpha_{t-1}} \mathbf{x}{t-2}+\sqrt{1-\alpha{t-1}} \mathbf{\epsilon}_{t-2}^\right)+\sqrt{1-\alpha_t} \mathbf{\epsilon}{t-1}^*\&=\sqrt{\alpha_t \alpha{t-1}} \mathbf{x}{t-2}+\sqrt{\alpha_t-\alpha_t \alpha{t-1}} \mathbf{\epsilon}{t-2}^*+\sqrt{1-\alpha_t} \mathbf{\epsilon}{t-1}^*\&=\sqrt{\alpha_t \alpha_{t-1}} \mathbf{x}{t-2}+\sqrt{{\sqrt{\alpha_t-\alpha_t \alpha{t-1}}}^2+\sqrt{1-\alpha_t}} \mathbf{\epsilon}{t-2}\&=\sqrt{\alpha_t \alpha{t-1}} \mathbf{x}{t-2}+\sqrt{\alpha_t-\alpha_t \alpha{t-1}+1-\alpha_t} \mathbf{\epsilon}{t-2}\&=\sqrt{\alpha_t \alpha{t-1}} \mathbf{x}{t-2}+\sqrt{1-\alpha_t \alpha{t-1}} \mathbf{\epsilon}{t-2}\&=\ldots\&=\sqrt{\prod{i=1}^t \alpha_i} \mathbf{x}0+\sqrt{1-\prod{i=1}^t \alpha_i \mathbf{\epsilon}_0}\&=\sqrt{\bar{\alpha}_t} \mathbf{x}_0+\sqrt{1-\bar{\alpha}_t} \mathbf{\epsilon}_0\end{aligned} $$

其中 $\left{\mathbf{\epsilon}_t^*, \mathbf{\epsilon}t\right}{t=0}^T \stackrel{\mathrm{iid}}{\sim} \mathcal{N}(\mathbf{\epsilon} ; \mathbf{0}, \mathbf{I})$。故我们可以得到 $q(\mathbf{x}_t \mid \mathbf{x}_0) = \mathcal{N}\left(\mathbf{x}_t ; \sqrt{\bar{\alpha}_t} \mathbf{x}_0,\left(1-\bar{\alpha}_t\right) \mathbf{I}\right)$。于是我们可以代入得到：

$$ q(\mathbf{x}_{t-1} \mid \mathbf{x}t,\mathbf{x}0) = \mathcal{N}(\mathbf{x}{t-1} ; \underbrace{\frac{\sqrt{\alpha_t}\left(1-\bar{\alpha}{t-1}\right) \mathbf{x}t+\sqrt{\bar{\alpha}{t-1}}\left(1-\alpha_t\right) \mathbf{x}_0}{1-\bar{\alpha}t}}{\mu_q\left(\mathbf{x}_t, \mathbf{x}0\right)}, \underbrace{\left.\frac{\left(1-\alpha_t\right)\left(1-\bar{\alpha}{t-1}\right)}{1-\bar{\alpha}t} \mathbf{I}\right)}{\mathbf{\Sigma}_q(t)} $$

为了让 $p_\theta(\mathbf{x}{t-1} \mid \mathbf{x}t)$ 尽可能接近真值 $q(\mathbf{x}{t-1} \mid \mathbf{x}t,\mathbf{x}0)$，我们也将其建模成高斯分布，即 $p\theta(\mathbf{x}{t-1} \mid \mathbf{x}t) = \mathcal{N}(\mathbf{x}{t-1};\mu\theta(\mathbf{x}_t,t),{\mathbf{\Sigma}_q(t)})$。两个高斯分布的 KL divergence 可以解析计算，即：

$$ D_{\mathrm{KL}}\left(q\left(\mathbf{x}_{t-1} \mid \mathbf{x}t, \mathbf{x}0\right) | p{\mathbf{\theta}}\left(\mathbf{x}{t-1} \mid \mathbf{x}t\right)\right)=\frac{1}{2 \sigma_q^2(t)}\left[\left|\mathbf{\mu}{\mathbf{\theta}}-\mathbf{\mu}_q\right|_2^2\right] $$

其中 $\mathbf{\Sigma}q(t)=\sigma_q^2(t) \mathbf{I}$。根据上面的公式，其实我们就是想让 $\mu\theta(\mathbf{x}_t,t)$ 尽可能匹配真值 $\mu_q(\mathbf{x}_t,\mathbf{x}0)$。由于 $\mu\theta(\mathbf{x}_t,t)$ 也是依赖于 $\mathbf{x}t$ 的，我们可以将 $\mu\theta$ 写成 $\mu_q$ 的形式，在这种情况下，就变成了给定 $\mathbf{x}_t$ 去预测 $\mathbf{x}_0$：

$$ \mathbf{\mu}{\mathbf{\theta}}\left(\mathbf{x}t, t\right)=\frac{\sqrt{\alpha_t}\left(1-\bar{\alpha}{t-1}\right) \mathbf{x}t+\sqrt{\bar{\alpha}{t-1}}\left(1-\alpha_t\right) \hat{\mathbf{x}}{\mathbf{\theta}}\left(\mathbf{x}_t, t\right)}{1-\bar{\alpha}_t} $$

进一步我们可以将 KL divergence 简化成：

$$ D_{\mathrm{KL}}\left(q\left(\mathbf{x}_{t-1} \mid \mathbf{x}t, \mathbf{x}0\right) | p{\mathbf{\theta}}\left(\mathbf{x}{t-1} \mid \mathbf{x}t\right)\right) = \frac{1}{2 \sigma_q^2(t)} \frac{\bar{\alpha}{t-1}\left(1-\alpha_t\right)^2}{\left(1-\bar{\alpha}t\right)^2}\left[\left|\hat{\mathbf{x}}{\mathbf{\theta}}\left(\mathbf{x}_t, t\right)-\mathbf{x}_0\right|_2^2\right] $$

Learning Diffusion Noise Parameters#

虽然我们手工构造 $\alpha_t$ 也可以构造出最终状态为标准正态分布的平稳马尔可夫链，但是我们也可以让一个网络去学一个最优构造。将优化目标展开可得到：

$$ \frac{1}{2 \sigma_q^2(t)} \frac{\bar{\alpha}_{t-1}\left(1-\alpha_t\right)^2}{\left(1-\bar{\alpha}t\right)^2}\left[\left|\hat{\mathbf{x}}{\mathbf{\theta}}\left(\mathbf{x}_t, t\right)-\mathbf{x}0\right|2^2\right] =\frac{1}{2}\left(\frac{\bar{\alpha}{t-1}}{1-\bar{\alpha}{t-1}}-\frac{\bar{\alpha}_t}{1-\bar{\alpha}t}\right)\left[\left|\hat{\mathbf{x}}{\mathbf{\theta}}\left(\mathbf{x}_t, t\right)-\mathbf{x}_0\right|_2^2\right] $$

令 $\text{SNR}(t) = \frac{\bar{\alpha}_t}{1-\bar{\alpha}t}$，由于 $\frac{1}{2 \sigma_q^2(t)} \frac{\bar{\alpha}{t-1}\left(1-\alpha_t\right)^2}{\left(1-\bar{\alpha}t\right)^2} > 0$，即 $\text{SNR}(t) - \text{SNR}(t-1) >0$， 则 $\text{SNR}(t)$ 是单调递减的。因此我们可以学习一个网络 $\exp (-\omega\eta(t))$ 去表示 $\text{SNR}(t)$。于是我们可以联合优化参数 $\eta$ 和 $\theta$。

在训练和推理中，我们会经常使用 $\bar{\alpha}_t$ 和 $1-\bar{\alpha}_t$，在这里我们有：

$$ \begin{aligned}&\frac{\bar{\alpha}_t}{1-\bar{\alpha}t}=\exp \left(-\omega{\boldsymbol{\eta}}(t)\right) \&\therefore \bar{\alpha}t=\operatorname{sigmoid}\left(-\omega{\boldsymbol{\eta}}(t)\right) \&\therefore 1-\bar{\alpha}t=\operatorname{sigmoid}\left(\omega{\boldsymbol{\eta}}(t)\right)\end{aligned} $$

Learn to predict the noise#

之前我们利用 reparameterization trick 得到了 $\mathbf{x}_t =\sqrt{\bar{\alpha}_t} \mathbf{x}_0+\sqrt{1-\bar{\alpha}_t} \epsilon_0$，反过来我们可以得到 $\mathbf{x}_0=\frac{\mathbf{x}_t-\sqrt{1-\bar{\alpha}_t} \mathbf{\epsilon}_0}{\sqrt{\bar{\alpha}_t}}$。带入 $\mu_q$ 可以得到：

$$ \mu_q(\mathbf{x}_t,\mathbf{x}_0)=\frac{1}{\sqrt{\alpha_t}} \mathbf{x}_t-\frac{1-\alpha_t}{\sqrt{1-\bar{\alpha}_t} \sqrt{\alpha_t}} \mathbf{\epsilon}_0 $$

因此我们同理可以将 $\mathbf{\mu}_{\mathbf{\theta}}\left(\mathbf{x}_t, t\right)$ 改写成：

$$ \mathbf{\mu}_{\mathbf{\theta}}\left(\mathbf{x}_t, t\right)=\frac{1}{\sqrt{\alpha_t}} \mathbf{x}_t-\frac{1-\alpha_t}{\sqrt{1-\bar{\alpha}t} \sqrt{\alpha_t}} \hat{\mathbf{\epsilon}}{\mathbf{\theta}}\left(\mathbf{x}_t, t\right) $$

对应的 KL divergence 就可以变成：

$$ D_{\mathrm{KL}}\left(q\left(\mathbf{x}_{t-1} \mid \mathbf{x}t, \mathbf{x}0\right) | p{\mathbf{\theta}}\left(\mathbf{x}{t-1} \mid \mathbf{x}_t\right)\right) = \frac{1}{2 \sigma_q^2(t)} \frac{\left(1-\alpha_t\right)^2}{\left(1-\bar{\alpha}_t\right) \alpha_t}\left[\left|\mathbf{\epsilon}0-\hat{\mathbf{\epsilon}}{\mathbf{\theta}}\left(\mathbf{x}_t, t\right)\right|_2^2\right] $$

至此已经证明，通过预测原始图像 $\mathbf{x}_0$ 来学习 VDM 等同于学习预测噪声。(Ho, Jain & Abbeel, 2020) 发现预测噪声会导致更好的性能，这里猜测是因为噪声均值为 0，因此更好学习。

Score-based Generative Models#

(Luo, 2022) 中通过 Tweedie’s Formula 架起了从 VAE 的视角看待 Diffusion Model 到从 score function 的视角看待 Diffusion Model 的桥梁。假设 $\mathbf{x} \mid \mu \sim \mathcal{N}(\mathbf{x};\mu,\sigma^2\mathbf{I})$，其中 $\sigma$ 是已知参数，$\mu \sim p(\mu)$ 是未知参数。Tweedie’s Formula 讲的是我们只需要边缘概率 $p(\mathbf{x})$，就可以知道均值的后验期望：

$$ \mathbb{E}[\mu \mid \mathbf{x}] = \mathbf{x} + \sigma^2 \nabla \log p(\mathbf{x}) $$

对于 $q\left(\mathbf{x}_t \mid \mathbf{x}_0\right)=\mathcal{N}\left(\mathbf{x}_t ; \sqrt{\bar{\alpha}_t} \mathbf{x}_0,\left(1-\bar{\alpha}_t\right) \mathbf{I}\right)$，我们利用 Tweedie’s Formula，只要给定样本 $\mathbf{x}_t$，我们就可以求得均值的后验期望为：

$$ \sqrt{\bar{\alpha}_t}\mathbf{x}_0=\mathbb{E}[\mu \mid \mathbf{x}_t]=\mathbf{x}_t + \sigma^2 \nabla \log p(\mathbf{x}_t) $$

就像对 $q\left(\mathbf{x}_t \mid \mathbf{x}_0\right)=\mathcal{N}\left(\mathbf{x}_t ; \sqrt{\bar{\alpha}_t} \mathbf{x}_0,\left(1-\bar{\alpha}_t\right) \mathbf{I}\right)$ 用 reparameterization trick 得到下面的表达式，随后类似地将 $\mathbf{x}_0$ 带入 $\mu_q$，至此证明通过预测原始图像 $\mathbf{x}0$ 来学习 VDM 等同于学习预测噪声并且等同于预测 score function $s\theta(\mathbf{x}_t,t)$。

$$ \mathbf{x}_t =\sqrt{\bar{\alpha}_t} \mathbf{x}_0+\sqrt{1-\bar{\alpha}_t} \epsilon_0 $$

我们可以发现，上面两个公式形式上完全一致，结合 Tweedie’s Formula 和 reparameterization trick 我们可以直接发现 score function 和噪声的关系：

$$ \nabla \log p(\mathbf{x}_t) = - \frac{1}{\sqrt{1- \bar{\alpha}_t} }\epsilon_0 $$

score function 衡量如何在数据空间中移动使得对数概率最大化。直观地讲，由于添加噪声 $\epsilon_0$ 会破坏图像，因此朝着相反的方向去噪就是更新对数概率的最优方向，上面的数学公式也证明了这一点。

Tweedie’s Formula 连接了 score-based model 和 DDPM，但是其不够直观。具体的讲解可以参照我的博客 Generative Modeling by Estimating Gradients of the Data Distribution 和其原文 (Generative Modeling by Estimating Gradients of the Data Distribution, Song, 2021)。

Guidance#

至此我们都专注于简单的数据生成 $p_\theta(\mathbf{x})$，然而我们经常需要一个带条件的数据生成 $p_\theta(\mathbf{x}|y)$，即我们希望通过条件信息 $y$ 去直接控制数据的生成。这也是现在 text2img、super-resolution 等模型的 backbone。

对于解码（逆过程）我们有：

$$ p_\theta\left(\mathbf{x}{0: T}\right)=p\left(\mathbf{x}T\right) \prod{t=1}^T p{\mathbf{\theta}}\left(\mathbf{x}_{t-1} \mid \mathbf{x}_t\right) $$

如果加上条件 $y$，我们可以很自然写出：

$$ p_\theta\left(\mathbf{x}{0: T} \mid y\right)=p\left(\mathbf{x}T\right) \prod{t=1}^T p{\mathbf{\theta}}\left(\mathbf{x}_{t-1} \mid \mathbf{x}_t, y\right) $$

因此我们能够像以前一样，对于不同的角度和实现，可以通过预测 $\hat{\mathbf{x}}\theta(\mathbf{x},t,y)$、$\hat{\epsilon}\theta(\mathbf{x}t,t,y)$ 或 $s\theta(\mathbf{x}_t,t,y)$ 来训练神经网络。但以这种方式训练的条件扩散模型有可能学会忽略任何条件信息的作用。因此 Guidance 作为一种更直接地控制模型给予条件信息权重的方法被提了出来。两种最流行的 Guidance 被称为 Classifier Guidance (Dhariwal & Nichol, 2021) 和 Classifier-Free Guidance (Ho & Salimans, 2022)。

Classifier Guidance#

从 score-based 的角度看待 Diffusion Model，因此 Guidance 的目标是学习 $\nabla \log p(\mathbf{x}_t \mid y)$。通过 Bayes rule，可以获得如下等式：

$$ \begin{aligned}\nabla \log p\left(\mathbf{x}_t \mid y\right) &=\nabla \log \left(\frac{p\left(\mathbf{x}_t\right) p\left(y \mid \mathbf{x}_t\right)}{p(y)}\right) \&=\nabla \log p\left(\mathbf{x}_t\right)+\nabla \log p\left(y \mid \mathbf{x}_t\right)-\nabla \log p(y) \&=\underbrace{\nabla \log p\left(\mathbf{x}t\right)}{\text {unconditional score }}+\underbrace{\nabla \log p\left(y \mid \mathbf{x}t\right)}{\text {adversarial gradient }}\end{aligned} $$

也就是我们的最终目标其实是学习一个和原来一样没有条件的的 score function 加上一个分类器 $p(y \mid \mathbf{x}_t)$。因此在 Classifier Guidance 中我们需要学习一个和之前一样的预测 score function 的网络，并且还需要学习一个输入任意时间步的噪声图片 $\mathbf{x}_t$ 预测其标签 $y$ 的分类器。因此在采样阶段，退火朗之万采样所用的 conditional score function 等于 unconditional score function 加上分类器的梯度。

为了更加精细得控制条件信息的权重，可以加入一个超参 $\gamma$，改写的 conditional score 为：

$$ \nabla \log p\left(\mathbf{x}_t \mid y\right)=\nabla \log p\left(\mathbf{x}_t\right)+\gamma \nabla \log p\left(y \mid \mathbf{x}_t\right) $$

当 $\gamma = 0$ 时，代表完全不考虑条件信息，此时和普通的生成模型没有区别。$\gamma$ 越大，模型生成的数据和条件越高度相关，但会因此减少生成模型的多样性，因此此时生成的数据仅限于和条件相关。

Classifier Guidance 的一个主要缺点是他需要额外训练一个分类器。因为该分类器对于任何加噪的图片都需要有分类能力，而目前与训练好的分类器都不能做到。因此该分类器也需要在训练 Diffuison Model 的时候一同训练。

Classifier-Free Guidance#

从 Classifier 到 Classifier-Free 的数学推导是非常巧妙的。对于 $\nabla \log p\left(\mathbf{x}_t \mid y\right)=\nabla \log p\left(\mathbf{x}_t\right)+\gamma \nabla \log p\left(y \mid \mathbf{x}_t\right)$，因为我们不想要多训练一个分类器，因此我们可以利用 Bayes rule 消掉 $\nabla \log p\left(y \mid \mathbf{x}_t\right)$：

$$ \begin{aligned}\nabla \log p\left(\mathbf{x}_t \mid y\right) &=\nabla \log p\left(\mathbf{x}_t\right)+\gamma\left(\nabla \log p\left(\mathbf{x}_t \mid y\right)-\nabla \log p\left(\mathbf{x}_t\right)\right) \&=\nabla \log p\left(\mathbf{x}_t\right)+\gamma \nabla \log p\left(\mathbf{x}_t \mid y\right)-\gamma \nabla \log p\left(\mathbf{x}_t\right) \&=\underbrace{\gamma \nabla \log p\left(\mathbf{x}t \mid y\right)}{\text {conditional score }}+\underbrace{(1-\gamma) \nabla \log p\left(\mathbf{x}t\right)}{\text {unconditional score }}\end{aligned} $$

到这里还是有两个网络，一个是不带条件的网络，一个是带条件的网络。训练两个网络也是一个麻烦的事情，所以我们可以直接把无条件的 $\nabla \log p(\mathbf{x}_t)$ 也当作是有条件的，但这个条件就是一个固定的值，比如 0。因此实际上，在训练的时候我们要对条件信息进行随机丢弃。Classifier-Free Guidance 是优雅的、并且也是现在的主流方案，因为它使我们能够更好地控制模型进行条件生成，并且除了训练一个的扩散模型外，不需要其他额外的任何东西。
